# Last Time

App Android per tenere traccia di quanto tempo è passato dall'ultima volta che è successo qualcosa (eventi personali, scadenze, abitudini).

Il design di riferimento è il mockup HTML in [`mokeup/last-time-mockup-v27.html`](mokeup/last-time-mockup-v27.html): tema scuro, card con contatori "da quanto tempo", vista alternativa a tabellone split-flap (stile Solari).

## Stato

Progetto in fase iniziale: per ora è presente solo il mockup.
